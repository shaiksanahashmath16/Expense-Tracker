# Expense-Tracker
#INTRODUCTION
Expense Tracker is a Python-based mini project developed to help users maintain a record of their daily expenses. Managing expenses is an important part of financial planning, and this application provides a simple and efficient way to monitor spending habits. The project allows users to add expenses, view all recorded expenses, calculate total expenditure, and exit the application through a menu-driven interface.This project is developed using Python programming language and demonstrates the use of fundamental programming concepts such as lists, loops, conditional statements, user input handling, and calculations.
#OBJECTIVE
The main objective of this project is to develop a simple application that enables users to record and track their expenses. The project also aims to improve programming skills by applying Python concepts in a real-world scenario
#PROBLEM STATEMENT
People often find it difficult to keep track of their daily expenses. Without proper records, it becomes challenging to analyze spending patterns and manage finances effectively. The Expense Tracker application provides a simple solution by allowing users to store and monitor their expenses in one place.

#WORKING OF THE PROJECT
1. The program starts by displaying a welcome message.
2. A menu containing available options is displayed.
3. The user selects an option.
4. If the user chooses Add Expense, the expense name and amount are stored.
5. If the user chooses View Expenses, all recorded expenses are displayed.
6. If the user chooses Show Total, the program calculates and displays the total spending.
7. If the user chooses Exit, the application terminates.
#ALGORITHM
Step 1: Initialize an empty list to store expenses.
Step 2: Display the main menu.
Step 3: Accept the user's choice.
Step 4: If the choice is Add Expense:
- Read expense name.
- Read expense amount.
- Store the expense in the list.
Step 5: If the choice is View Expenses:
- Display all stored expenses.
Step 6: If the choice is Show Total:
- Calculate the sum of all expense amounts.
- Display the total spending.
Step 7: If the choice is Exit:
- Terminate the program.
Step 8: Repeat the process until the user exits.
